## 2. 設計思想

### 5つの設計原則

| 原則 | 内容 |
|---|---|
| **① 安定性最優先** | クラッシュしない。NAS接続断もクラッシュではなく「状態」として扱う |
| **② 触らなくても快適** | デフォルト設定で十分快適に読める |
| **③ 触ればもっと自分好みに** | 設定を深く潜れば細かくカスタマイズできる |
| **④ よく使う設定はプリセットで一発** | 複数のプリセットを保存・即時切り替え |
| **⑤ UIは自分好みにカスタマイズできる** | ツールバーの中身・表示タイミングを変更可能 |

### NASファーストの思想

```
NASもローカルも「場所」として統一的に扱う
ファイルは参照するだけ・汚さない・移動させない
メタデータ（評価・タグ等）はアプリ内DBで管理
サムネイルのみキャッシュとして内部に保存
```

### Comic Glassリスペクト

本アプリはiOS用Comic Glassに強くインスパイアされている。
UIの設計思想・操作概念を参考にしており、アプリ説明欄にその旨を明記する。

---

### ① ホーム画面

#### 上部メニュー
```
[HOME]　　　　　　　　　[🔍] [名前▼] [≡⊞] [編集]
　　　　　　　　　　　　検索  ソート  表示切替  編集モード
```

#### ファイル・フォルダ表示エリア
- アプリ専用保存フォルダ内のコミックファイル
- ショートカット登録したリモートフォルダ
- お気に入り登録したコミックファイル

#### 下部メニュー
```
[A:About] [B:サーバー] [C:プリセット] [D:明るさ] [E:設定]
```

| ボタン | 内容 |
|---|---|
| **A：About（仮称）** | Homeに戻る／About表示／ストレージ使用量／閲覧履歴／このフォルダの全サムネイルを再作成 |
| **B：リモートサーバー** | 登録サーバー一覧・新規追加／転送状況／サーバー関連設定 |
| **C：プリセット** | プリセット選択・切り替え |
| **D：明るさ** | スライドバーで明るさ調整 |
| **E：設定** | 各種設定にアクセス |

#### 検索・ソート（引き出し式）
- 🔍タップで検索バーが展開
- ソートボタンで名前／日付／ステータス／ユーザー設定を切り替え

---

### ② フォルダ・ファイルブラウザ画面（Home以外）

#### 上部メニュー
```
[←] [現在のフォルダ位置]　　　[🔍] [名前▼] [≡⊞] [編集]
　長押しでサーバールートまたはHomeに戻るボタン表示
```

#### 下部メニュー

**ローカルフォルダ時：**
```
[A:About] [C:プリセット] [D:明るさ] [E:設定]
```

**リモートフォルダ時：**
```
[A:About] [B:サーバー] [F:DL/STR切替] [D:明るさ] [E:設定]
```

| ボタン | 内容 |
|---|---|
| **F：DL/STR切替** | ダウンロードモード／ストリーミングモードの切り替え |

#### 現在のフォルダ内ファイルを一括ダウンロードするボタン
- 上部メニュー下またはファイルエリア上部に配置
- ダウンロード／ストリーミングモードに関わらず表示

### ④ 閲覧画面（タップ後・メニュー表示）

#### タップ後の演出
- ページ表示がやや暗くなる（ふわっとアニメーション）
- 上部・下部メニューがスライドして出現

#### 中央に表示される閲覧モードメニュー
```
[閉じるボタン]　← 本を閉じてブラウザ画面に戻る

[あ:画面ロック] [い:スライドショー] [う:読書位置切替] [え:メニュー展開]

展開後の追加メニュー：
・音量ボタンで移動
・マイク音量で移動
・ページ画像を保存
・次の本へ
・最初のページへ移動
・前の本へ
・表紙に設定
```

#### 下部メニュー（閲覧時専用）
```
[A:ブックマークメニュー] [B:ブックマーク] [C:プリセット] [D:明るさ] [E:設定]

ページ移動スライダー：　████████░░░　34/120
ページ微調整ボタン：　　◀ ▶
```

| ボタン | 内容 |
|---|---|
| **A：ブックマークメニュー** | 登録済みブックマークをサムネイル付きで表示・削除・全削除 |
| **B：ブックマーク** | 現在ページをブックマーク登録（登録済みは色変化） |
| **C：プリセット** | プリセット選択・切り替え |

### ファイル・フォルダの表示仕様

#### コミックファイルの表示

```
┌──────────────────────────────────────┐
│ [表紙  ] タイトル名                   │
│ サムネイル 作者名                     │
│         フォルダ名                    │
│         10/200　昨日　2026/02/10     │
│ [A] [Cached] [Ⓘ]                    │
└──────────────────────────────────────┘
```

| 要素 | 内容 |
|---|---|
| タイトル・作者名 | ファイル名から自動抽出（後述のパターン参照） |
| ページ数表示 | 現在閲覧ページ／総ページ数 |
| 最終閲覧日 | 今日／昨日／〇日前（〜100日）／日付 |
| A | プリセット番号（設定時のみ）・ブックマーク（保存時のみ） |
| Cached | ローカルキャッシュがある場合のみ表示 |
| Ⓘ | タップでファイル情報ポップアップ |

#### ファイル情報ポップアップ（Ⓘタップ時）
- ファイル名
- ファイルサイズ
- 更新日
- 状態：未読／読書中／既読（タップで切り替え）
- 個別設定：プリセットの割り当て
- キャッシュ消去
- サムネイルを再作成

#### ファイル名からのタイトル・作者名抽出

```
対応パターン：
(カテゴリ) [作者名] タイトル.zip

例：(一般コミック) [桂正和] ウイングマン1巻.zip
→ 作者名：桂正和
→ タイトル：ウイングマン1巻
→ (カテゴリ)部分は非表示

パターン不一致の場合：ファイル名をそのまま表示
```

#### フォルダの表示
- フォルダ種別をアイコン／バッジで区別
  - ローカルフォルダ
  - リモート（NAS）フォルダ
  - ショートカット

---

### コミックファイルをタップ時のポップアップ

```
・○○ページから再開（過去に開いている場合）
・最初から読む
・キャッシュを消去（ストリームファイルの場合）
```

**設定で「ポップアップなしで即再開」に変更可能**

---

### 編集モード

#### 発動条件
- 上部メニューの「編集」ボタンをタップ

#### 編集モード時の表示変化
- ファイル・フォルダにチェックボックスが出現
- 下部メニューが編集専用セットに切り替わり

#### 編集モード下部メニュー

**ローカルフォルダ時：**
```
[全選択] [削除] [移動] [フォルダ作成]
ファイル未選択時：削除・移動はグレーアウト
```

**リモートフォルダ時：**
```
[全選択] [ダウンロード] [フォルダ作成]
削除・移動は表示しない（リモートファイルの安全性確保）
```

## 5. 機能仕様

### ストレージ・ファイルアクセス

#### 「場所」の種類

| 種類 | 例 | 特徴 |
|---|---|---|
| NAS（SMB接続） | //192.168.1.10/manga/ | ストリーミング or DL |
| ローカルフォルダ | /sdcard/Comics/ | 直接参照 |
| アプリ専用フォルダ | 内部ストレージ/ComicVeil/ | 転送ファイルの保管場所 |

全ての「場所」をブラウザ画面で同一のUIで操作する。

#### 内部ストレージの使い方

```
保存するもの：
・SQLiteデータベース（評価・タグ・読書位置・プリセット・設定）
・サムネイルキャッシュ
・アプリ設定

保存しないもの：
・コミックファイル本体（参照のみ）
```

---

### サムネイル管理（3段階）

| 操作 | 対象 | アクセス方法 |
|---|---|---|
| 個別再作成 | 1ファイル | ファイルをタップ → ポップアップ |
| フォルダ内再作成 | 現在のフォルダのみ | Aboutメニュー |
| 全体再作成 | ライブラリ全体 | Settings → ライブラリ管理 |

---

### ファイル転送

| 方法 | Phase | 説明 |
|---|---|---|
| USB転送（PC→スマホ） | 1 | アプリ側対応不要 |
| 他アプリからの共有（Intent） | 1 | 「Comic Veilで開く」で受け取り |
| NASから直接ストリーミング | 1 | 転送不要 |
| Wi-Fi経由転送UI | 2 | アプリ内Webサーバー機能 |

### プリセット機能

#### プリセットに含まれる設定項目

**頻繁に変える項目（プリセットの核心）：**
- ページ送り方向（右綴じ／左綴じ）
- トリミング（ON/OFF・強度）
- ページ間隔
- 明るさ

**たまに変える項目：**
- 表示モード（単ページ／見開き／縦スクロール）
- 背景色（黒／白／グレー）
- コントラスト
- バウンスアニメーション（ON/OFF）
- アニメーション速度

**詳細設定（隠し項目）：**
- プリロードページ数
- タップゾーン設定
- アニメーション種類

#### プリセット数
- A〜E（仮）の5つ（上限は要検討）

---

### タグ・評価機能（Lightroomライク）

#### 評価
- ★1〜5の5段階

#### カラーラベル
- 赤／黄／緑／青／紫
- デフォルトの名前定義あり（変更可能）
  - 例：赤＝未読、緑＝読了、青＝お気に入り

#### フラグ
- ピック（✓）：絶対残す
- リジェクト（✗）：削除候補
- なし

#### タグ（キーワード）
- 自由入力・複数付与可能
- 例：「SF」「完結」「再読したい」「作者名」

#### 絞り込み・フィルタリング
- ★評価でフィルタ
- カラーラベルでフィルタ
- フラグでフィルタ
- タグのAND/OR検索
- 未評価ファイルのみ表示

#### 表示・非表示の切り替え
- 設定でデフォルトを設定
- ブラウザ画面でも即時切り替え可能

---

### ホーム画面のショートカット・お気に入り

| 機能 | 内容 |
|---|---|
| ショートカット | フォルダをピン留め。どの「場所」でも登録可 |
| お気に入り | 個別ファイルを登録。どの「場所」のファイルでも登録可 |

---

### 読書位置の自動保存

- 閲覧中は常にページ位置をDBに保存
- アプリがクラッシュしても再起動時に再開可能
- 「○○ページから再開」ポップアップで表示

## 6. UI・UX仕様

### アニメーション

#### ページ送りアニメーション
- **デフォルト**：スライド＋スプリング（バウンス）
- バウンスはComic GlassのデフォルトON状態を参考に実装
- SpringAnimationで実装（dampingRatio・stiffness調整可能）

**設定項目：**
- アニメーション種類：スライド／フェード／なし
- バウンス：ON/OFF
- アニメーション速度：速い／普通／遅い

#### メニューのスライドアニメーション
- 閲覧モード移行時：上メニューは上へ、下メニューは下へスライドして消える
- タップ時：下メニューが下からスライドして現れる
- AnimatedVisibility（Jetpack Compose）で実装

#### 画面を暗くする演出
- タップ後にページ表示がふわっと暗くなる
- Boxの上にBlackをalpha 0.0→0.3でアニメーション

```
基本設定（誰でも触る）：常に表示
詳細設定（こだわる人向け）：「さらに〇項目を表示」で展開
```

### ツールバーカスタマイズ

**表示タイミング：**
- タップで出現（デフォルト）

**ツールバーの中身：**
- よく使うボタンを選んで配置

## 7. データ設計

### テーブル構成（概要）

```sql
-- 場所の管理
locations
  id / 表示名 / タイプ(NAS/local/internal) / パス / 認証情報

-- ファイルのメタデータ
files
  id / location_id / 相対パス / タイトル / 作者名
  評価 / カラーラベル / フラグ
  最終閲覧日 / 現在ページ / 総ページ数
  状態(未読/読書中/既読) / 登録日

-- タグ
tags
  id / 名前

-- ファイルとタグの紐付け
file_tags
  file_id / tag_id

-- ブックマーク
bookmarks
  id / file_id / ページ番号 / 登録日

-- プリセット
presets
  id / 名前（A〜E）/ 設定JSON

-- サムネイルキャッシュ
thumbnails
  id / file_id / キャッシュパス / 生成日
```

### ファイル識別の方針

```
NASのIPアドレスや共有名が変わっても追跡できるよう
「場所の表示名（location.表示名）+ 相対パス」で管理
パス絶対値には依存しない
```

## 8. 対応フォーマット

### 画像フォーマット

| カテゴリ | フォーマット | 優先度 |
|---|---|---|
| Web主流 | JPEG, PNG, WebP, GIF | 最優先 |
| 次世代 | HEIC, AVIF, JPEG XL | 高 |
| AI生成 | PNG（tEXtチャンク）, WebP | Phase 2 |

### アーカイブフォーマット

| 形式 | 拡張子 | ライブラリ |
|---|---|---|
| ZIP | .zip / .cbz | zip4j |
| RAR | .rar / .cbr | junrar |
| 7-Zip | .7z | Apache Commons Compress |
| PDF | .pdf | Android標準 PdfRenderer |

---

### AI生成画像メタデータ対応（Phase 2）

| 生成ツール | 埋め込み場所 | 表示内容 |
|---|---|---|
| Stable Diffusion / A1111 | PNG tEXt `parameters` | プロンプト・ネガティブ・Steps・Sampler・CFG・Seed・Model |
| ComfyUI | PNG tEXt `workflow` | 全ノード情報（JSON） |
| NovelAI | PNG tEXt | 独自形式 |
| ChatGPT / Gemini | C2PA | 「AI生成」バッジ表示のみ |
